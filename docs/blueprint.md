# **App Name**: Committed Bodies Hub

## Core Features:

- User Authentication: User authentication (login/signup) using Firebase Authentication
- Client Dashboard: Client dashboard showing gym stats for the week (pulled from Firebase)
- Meal Booking System: Meal booking system for users (data saved in Firebase)
- Admin Dashboard: Admin dashboard to manage users, bookings, and menus (with full CRUD operations)
- Menu Management: Menu management to update meals available for booking

## Style Guidelines:

- Primary color: Rose quartz (#F7CAC9) for a calming and approachable feel.
- Background color: Light gray (#333333) provides a modern, dark-themed contrast, creating an atmospheric but highly legible effect.
- Accent color: Burnt orange (#D36B36) for call-to-action buttons and highlights to draw attention.
- Body and headline font: 'Inter' (sans-serif) provides a clean, modern look for readability.
- Use minimalistic icons related to health, fitness, and user profiles.
- Clean, modern layout with a focus on data visualization and easy navigation.
- Subtle animations for transitions and feedback, such as loading stats and confirming actions.
